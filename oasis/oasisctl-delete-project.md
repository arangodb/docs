# Oasisctl Delete Project

Delete a project the authenticated user has access to

## Synopsis

Delete a project the authenticated user has access to

```
oasisctl delete project [flags]
```

## Options

```
  -h, --help                     help for project
  -o, --organization-id string   Identifier of the organization

```
