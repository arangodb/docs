# Oasisctl Lock Deployment

Lock a deployment, so it cannot be deleted

## Synopsis

Lock a deployment, so it cannot be deleted

```
oasisctl lock deployment [flags]
```

```

  -h, --help                     help for deployment
  -o, --organization-id string   Identifier of the organization
```
