# Oasisctl Delete Backup Policy

Delete a backup policy for a given ID.

## Synopsis

Delete a backup policy for a given ID.

```
oasisctl delete backup policy [flags]
```

```

  -i, --id string                Identifier of the backup policy
  -o, --organization-id string   Identifier of the organization
```
