# Oasisctl Top

Show the most important server metrics

## Synopsis

Show the most important server metrics

```
oasisctl top [flags]
```

## Options

```
  -d, --deployment-id string     Identifier of the deployment
  -h, --help                     help for top
  -o, --organization-id string   Identifier of the organization
  -p, --project-id string        Identifier of the project
```
